# Renderer Components Instructions

This directory contains React components organized by feature.

```

│   ├── Loader.tsx
│   ├── Tooltip.tsx

│   └── TitleBar.tsx      # Custom window title bar
├── Sidebar/              # Left navigation panel
├── MainContent/          # Main content area
├── Modal/                # Modal dialogs
│   ├── GlobalModal.tsx   # Game detail modal
│   ├── ConfirmModal.tsx  # Confirmation dialogs
│   └── PromoModal.tsx    # Promotional modals
├── Settings/             # Settings panel
├── Notifications/        # Toast notifications
├── UpdateNotification/   # App update banner
├── GamepadHints/         # Steam Deck gamepad hints
├── Icons/                # SVG icon components
├── Elements/             # Shared element components
├── AppLoader/            # Initial loading screen
└── Placements/           # Banner ad placements
```

## Component Patterns

### Feature Folder Structure

```
components/
└── FeatureName/
    ├── FeatureName.tsx       # Main component
    ├── FeatureItem.tsx       # Sub-component
    ├── FeatureName.module.css # Scoped styles (if needed)
    ├── types.ts              # Local type definitions
    ├── hooks.ts              # Feature-specific hooks
    └── index.ts              # Re-exports
```

### Named Exports

```typescript
// FeatureName.tsx
export const FeatureName: React.FC<Props> = ({ ... }) => {
  return <div>...</div>;
};

// index.ts
export { FeatureName } from './FeatureName';
export type { Props as FeatureNameProps } from './FeatureName';
```

## UI Components (`ui/`)

Reusable, stateless components:

```typescript
// Button usage
<Button variant="primary" onClick={handleClick}>
  Click me
</Button>

// Input with label
<Input 
  label="Search" 
  value={query} 
  onChange={(e) => setQuery(e.target.value)} 
/>

// Toggle switch
<Switch 
  checked={enabled} 
  onChange={setEnabled} 
  label="Enable feature" 
/>
```

## Styling

### TailwindCSS

```tsx
// Use Tailwind classes
<div className="flex items-center gap-2 p-4 bg-gray-800 rounded-lg">
  <span className="text-white font-medium">{title}</span>

```

### Dark Theme

App uses dark theme exclusively:
- Backgrounds: `bg-gray-800`, `bg-gray-900`, `bg-black`
- Text: `text-white`, `text-gray-300`, `text-gray-400`
- Accents: `text-cyan-400`, `bg-cyan-600`

### Animations

Use framer-motion, respect user preference:

```tsx
import { motion } from 'framer-motion';
import { useSettingsStore } from '@store/useSettingsStore';

const Component = () => {
  const { animationsEnabled } = useSettingsStore();
  
  return (
    <motion.div
      initial={animationsEnabled ? { opacity: 0, y: 10 } : false}
      animate={{ opacity: 1, y: 0 }}
      transition={{ duration: 0.2 }}
    >
      Content
    </motion.div>
  );
};
```

## Modal Pattern

```tsx
import { useModalStore } from '@store/useModalStore';

// Open modal
const { openModal } = useModalStore();
openModal({ type: 'game', gameId: game.id });

// Modal component
const Modal = () => {
  const { isOpen, modalData, closeModal } = useModalStore();
  
  if (!isOpen) return null;
  
  return (
    <div className="modal-overlay" onClick={closeModal}>
      <div className="modal-content" onClick={(e) => e.stopPropagation()}>
        {/* Content */}
      </div>
    </div>
  );
};
```

## Gamepad Mode (Steam Deck)

Components should support gamepad navigation:

```tsx
import { useGamepadModeStore } from '@store/useGamepadModeStore';

const Component = () => {
  const { isGamepadMode } = useGamepadModeStore();
  
  return (
    <button 
      className={cn(
        'base-styles',
        isGamepadMode && 'larger-hit-target focus-visible:ring-2'
      )}
    >
      Action
    </button>
  );
};
```

## Performance Guidelines

1. **Memoize callbacks** passed to children
2. **Use React.lazy** for large components
3. **Virtualize long lists** (`@tanstack/react-virtual`, see `Sidebar/GameList.tsx`)
4. **Avoid inline objects** in JSX (causes re-renders)

```tsx
// AVOID
<Component style={{ color: 'red' }} /> // New object every render

// BETTER
const style = useMemo(() => ({ color: 'red' }), []);
<Component style={style} />
```
