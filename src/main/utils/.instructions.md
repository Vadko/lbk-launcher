# Main Process Utilities Instructions

This directory contains utility functions for the main process.

## Structure

```
src/main/utils/
├── platform.ts          # OS detection (isWindows, isLinux, isMacOS)
├── files.ts             # File path handling, Cyrillic transliteration
├── logger.ts            # Logging utilities with timing
├── open-external.ts     # Open URLs in default browser
├── steam-launcher.ts    # Start/stop Steam client
├── steam-launch-options.ts # Set Steam launch options
├── steam-cef.ts         # Steam CEF (Chromium) integration
├── cef-flag-file.ts     # CEF flag file management
├── vdf-parser.ts        # Steam VDF file parsing
├── store-storage.ts     # Electron store persistence
├── theme.ts             # System theme detection
├── epic-launcher.ts     # Epic Games Launcher integration
├── heroic-launcher.ts   # Heroic Launcher integration
└── lutris-launcher.ts   # Lutris integration
```

## Key Utilities

### platform.ts

```typescript
import { isWindows, isLinux, isMacOS, getPlatform } from './platform';

if (isWindows()) { /* Windows-specific code */ }
if (isLinux()) { /* Linux-specific code */ }
if (isMacOS()) { /* macOS-specific code */ }

const platform = getPlatform(); // 'windows' | 'linux' | 'macos'

// macOS Liquid Glass support
if (supportsMacOSLiquidGlass()) {
  // macOS 26+ only
}
```

### files.ts

Handle Cyrillic filenames for installers:

```typescript
import { renameFileToTranslit, getTransliteratedPath } from './files';

// Get transliterated path (without renaming)
const safePath = getTransliteratedPath('/path/to/Файл.exe');
// Returns: '/path/to/Fail.exe'

// Rename file to transliterated name
const newPath = renameFileToTranslit('/path/to/Файл.exe');
// Renames file and returns new path
```

### logger.ts

```typescript
import { createTimer, initLogger } from './logger';

// Measure operation time
const timer = createTimer('Operation name');
await doSomething();
timer.end(); // Logs: "[Operation name] completed in 123ms"
```

### steam-launch-options.ts

```typescript
import { writeSteamLaunchOptions } from './steam-launch-options';

// Set Steam launch options for game
await writeSteamLaunchOptions(steamAppId, 'LANG=uk_UA.UTF-8 %command%');
```

### open-external.ts

```typescript
import { openExternalUrl } from './open-external';

```

## Platform-Specific Patterns

### Registry Access (Windows)

```typescript
import { execSync } from 'child_process';

if (isWindows()) {
  const output = execSync('reg query "HKLM\\SOFTWARE\\..."').toString();
}
```

### Process Spawning

```typescript
import { spawn } from 'child_process';

// Windows .bat files
if (isWindows() && file.endsWith('.bat')) {
  spawn(file, [], { shell: true, cwd: dir });
}

// Linux with environment
if (isLinux()) {
  spawn(file, [], { 
    env: { ...process.env, STEAM_COMPAT_DATA_PATH: prefix }
  });
}
```

### Path Handling

```typescript
import path from 'path';

// Use path.join for cross-platform paths
const fullPath = path.join(basePath, 'subfolder', 'file.txt');

// Normalize separators
const normalized = path.normalize(inputPath);
```

## Steam Integration

### VDF Parsing

```typescript
import { parse } from 'fast-vdf';

const vdfContent = fs.readFileSync(libraryFoldersPath, 'utf8');
const parsed = parse(vdfContent);
```

### Launch Options

Steam launch options are written to `localconfig.vdf`:
- Location: `Steam/userdata/{userId}/config/localconfig.vdf`
- Requires Steam restart to take effect

## Guidelines

1. **Always check platform** before using platform-specific APIs
2. **Handle Cyrillic paths** - Many Ukrainian files have Cyrillic names
3. **Use async operations** when possible to avoid blocking
4. **Log with module prefix**: `console.log('[Utils] ...')`
5. **Test on all platforms** for any path or process operations
